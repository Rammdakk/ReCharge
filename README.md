
# Recharge - Клиентская версия приложения, для записи пользователя на занятия

Данный проект является частью [дипломной работы](https://www.hse.ru/ba/se/students/diplomas/page8.html/924181757), включающей в себя клиентскую и серверную часть.
Клиентская часть представлена приложением клиента и приложением продавца.

## Тестирование приложения
Отправка смс на серверной части не включена. Для тестового входа вместо кода из смс 12345.
